# ServiceHub

ServiceHub — внутрішній веб-інструмент для швидкого пошуку інформації про послуги компанії. Проєкт є **MVP (Minimum Viable Product)**, створеним для тестування і демонстрації основного функціоналу пошуку та управління контентом у команді продажів.

---

## Проєкт

Проєкт написаний на **Django** з використанням **PostgreSQL** та **Docker Compose**. Для пошуку використовується **повнотекстовий пошук**, а також реалізована **локалізація на три мови**: українська, російська, англійська.  

Основна мета проєкту — надати менеджерам з продажів **швидкий доступ до інформації про послуги**, пакети, ключові переваги та FAQ, щоб вони могли оперативно відповідати клієнтам без потреби шукати інформацію в окремих файлах чи чатах.

При першому запуску проєкт автоматично:

- Заповнює базу даних **тестовими даними** (послуги, пакети, FAQ).
- Створює користувача-адміністратора для доступу до стандартної **адмінпанелі Django**, де можна додавати та редагувати матеріали.
- Забезпечує готовий для роботи MVP інтерфейс із можливістю миттєвого пошуку та фільтрації контенту.

---

## Запуск локально

1. Клонувати репозиторій:

```bash
git clone git@github.com:Paprics/ServiceHub.git
cd servicehub
````

2. Запустити контейнер через Docker Compose:

```bash
docker compose up --build
```

3. Відкрити браузер за адресою:

```
http://127.0.0.1:8000/       # веб-інтерфейс
http://127.0.0.1:8000/admin  # адмінпанель Django
```

* Логін та пароль адміністратора: створюються автоматично при першому запуску.

  * login: admin
  * password: 1234

---

## Адмінпанель

* Додавання та редагування послуг відбувається через **стандартну адмінпанель Django**.
* Всі матеріали зберігаються у базі PostgreSQL та доступні для миттєвого пошуку через веб-інтерфейс.

---

## Стек технологій

* **Backend:** Django
* **База даних:** PostgreSQL
* **Контейнеризація:** Docker Compose
* **Пошук:** повнотекстовий
* **Локалізація:** українська, російська, англійська
